# Operational Governance Truth Example

## Situation

A governance document lives outside the nominal runtime-active directory.

However:

- the active path explicitly routes into it
- it defines current execution boundaries or coordination rules
- ignoring it would make current work unsafe

## Wrong Response

Treat it as reference-only because of where it sits in the tree.

That confuses physical location with operational role.

## Better Response

- classify it by function first
- mark it as current operational authority
- if it conflicts with higher-surface routing, hand it off through authority triage

## Why This Matters

In governance-heavy repositories, some current truth is operational rather than thematic. The skill must preserve that distinction.
