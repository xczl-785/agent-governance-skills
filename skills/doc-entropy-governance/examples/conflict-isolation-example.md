# Conflict Isolation Example

## Situation

Two documents describe the same mechanism:

- one appears newer
- one appears more authoritative
- neither can be mechanically proven as the true current source

## Wrong Response

Choose one by intuition and edit the other to match.

That may create apparent consistency while hiding unresolved authority conflict.

## Better Response

- do not rewrite either document's core claim automatically
- add a standardized governance state if appropriate
- add truth-source pointers where safe
- reduce active-path exposure if one document should not keep guiding current work
- emit a conflict summary or decision item

## Why This Matters

The skill must lower misleading risk without pretending semantic judgment has already been made.
